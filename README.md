## 🤝 介绍

这是一个专业的戳一戳插件，机器人被戳时，会随机触发这些回复事件（反戳：LLM：face：表情包：禁言：命令）， 也支持命令调用、关键词触发，触发的概率权重可自定义，文本，表情包等都可自定义，当然也能开箱即用！

## 📦 安装

在astrbot的插件市场搜索astrbot_plugin_pokepro，点击安装即可`

## ⌨️ 使用说明

### 命令表

|     命令      |                    说明                    |
|:-------------:|:-----------------------------------------------:|
| {被戳}   | 触发回复事件（反戳：LLM：face：表情包：禁言：命令） |
| 戳 次数@XXX | 戳指定人，可指定次数，不填则默认为1  |
| 戳我   |   戳你自己（你是m吧？）  |
| 戳全体成员   |   戳全体成员，超过200人的群会随机抽取200人来戳  |
